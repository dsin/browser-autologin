browser-autologin
==========
autologin into chrome or firefox

Installation
-------------------------
`git clone git@github.com:dsin/browser-autologin.git`

- Google Chrome
Steps
1. open `chrome://extensions/`
2. enable `developer mode`
3. click `load unpacked`
4. pick the folder that contains `autologin.user.js` and `manifest.json`

- Firefox

1. need `Greasemonkey` add-on
2. drag and drop to firefox page, firefox will pop-up to ask for installation
